# 5-Question Quiz + Answers — How Bitcoin Mining Works

*(Generated using the Quiz Generation prompt template; answers added afterward using the Feedback template's logic.)*

**1. What is the "block reward" a miner receives for successfully mining a block?**
Answer: Newly created ("minted") bitcoin, plus the transaction fees from that block — awarded to the first miner to solve the puzzle.

**2. What is the name for the number miners repeatedly change while trying to find a valid hash?**
Answer: The **nonce** — a value miners change over and over so the resulting hash is different each try.

**3. Why does Bitcoin mining use so much electricity?**
Answer: Because miners are making trillions of random guesses per second (brute-force, "proof-of-work") to be the first to find a valid hash — not because of data storage.

**4. What happens to the mining reward roughly every four years?**
Answer: It's cut in half — an event called the **"halving."**

**5. True or False: A miner can predict in advance which number will produce a valid hash.**
Answer: **False.** Hashing is one-way and unpredictable, so miners must try numbers essentially at random until one happens to work.
